Introduction à la programmation web

TP n°1 : Le langage HTML

Objectifs :

L'objectif de ce tp est de découvrir le langage HTML au travers de la création d'un mini site web. Vous pouvez travailler indifféremment sous Windows ou Linux.

Pour comprendre l'utilisation des balises HTML utilisées dans ce TP les liens étiquetés en vert vous renvoient vers la page correspondante dans le tutoriel du w3schools consacré à HTML (http://www.w3schools.com/html).

Préambule

Avant toute chose il vous est recommandé de créer un répertoire IPW dans lequel vous pourrez ranger les différents travaux que vous réaliserez au cours des séances de TP. Il vous est également conseillé de créer un sous répertoire (TP1, TP2, ...) consacré à chaque TP.

Exercice 1: Création d'une page CV

  1. Créez un fichier HTML portant le nom monCV.html. Ce fichier sera composé des éléments suivants :
    • une partie en tête (balise head) dans laquelle vous définirez le titre de votre page (balise title),
    • une partie corps (balise body),
    • à l'intérieur du corps de votre page un premier titre (balise h1).
  2. Créez un paragraphe (balise p) sous le titre et insérez votre photographie dans celui-ci (balise img) ainsi que vos informations d'état civil (nom, prénom, date de naissance, ...).

    Si vous ne disposez pas encore d'une photographie personnelle sous forme numérique vous pouvez, en attendant, utiliser l'image ci-dessous que vous pourrez enregistrer sur votre compte par un clic droit avec la souris :

  3. Ajoutez les sections suivantes :
    • Diplômes
    • Expériences professionnelles
    • Activités et loisirs
    Chaque section sera délimitée par une balise (balise section) et débutera par son titre (balise h2).
  4. Complétez au moins l'une des sections précédentes en listant ses éléments sous forme de liste à puces (balise ul).

Exercice 2 : Tables

  1. Créez un second fichier HTML (agenda.html)
  2. Dans ce second document présenter votre emploi du temps hebdomadaire sous la forme d'un tableau (balise table). Ce tableau comportera une colonne par jour et une ligne par tranche horaire de cours.

Exercice 3 : Liens hypertextes

  1. Afin de pouvoir remonter directement au début de votre CV, ajoutez une ancre (balise a avec attribut id) au début de votre document et une référence à cette ancre à la fin de votre document (fichier monCV.html).
  2. Pour mettre en place un sommaire dans votre page CV:
    • ajouter une ancre avant chaque titre de section,
    • ajouter une section de titre Sommaire après le titre de votre CV,
    • dans cette section ajoutez une liste de références à ces différentes ancres.
  3. Mettez en place des liens de navigation qui permettent de passer directement de la page monCV.html à la page agenda.html et inversement.

Exercice 4 : Image réactive

Il s'agit de doter votre mini site d'une page d'accueil (index.html) présentant un menu permettant de choisir entre :

  1. la page d'accueil de l'IMSS de l'UFR SHS,
  2. la page d'accueil du master IC2A,
  3. votre CV.

Ce menu est réalisé à l'aide d'une image réactive.

Affichage de l'image

Récupérez l'image menu.png et créez une page index.html contenant:

  • le titre Menu,
  • cette image.

Rendre l'image réactive

Rendez l'image réactive (balise map) en définissant trois zones associées aux URL suivantes :

  • http://imss.upmf-grenoble.fr/ pour le lien vers l'IMSS de l'UFR SHS,
  • http://imss-www.upmf-grenoble.fr/master-ic2a/ pour le lien vers le master IC2A,
  • votreNom.html pour votre CV.

Il existe de nombreux outils pour construire des images réactives. Parmi ceux-ci on peut noter GIMP (GNU Image Manipulation Program), un logiciel gratuit offrant de nombreuses et puissantes fonctionalités de manipulation d'image dont en ce qui nous concerne la création d'images réactives (voir le tutorial How To Create An Image Map Using GIMP de Tim Lenahan).

Permettre la navigation dans le site

Pour faciliter la navigation dans le site, ajoutez des liens sur les pages monCV.html et agenda.html qui permettent de retourner directement sur la page d'accueil.
